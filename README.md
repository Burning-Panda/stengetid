# Stengetid

Simple app to show todays closing time of [Vinmonopolet](https://www.vinmonopolet.no/).

## Development

```node
yarn install
```

### Compiles and hot-reloads for development

```node
yarn serve
```

### Compiles and minifies for production

```node
yarn build
```

### Lints and fixes files

```node
yarn lint
```
